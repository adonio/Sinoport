## 2. 目标

把“新增站点接入”收口成一套可执行、可验收、可回滚的标准 SOP。

本版范围：
- 站点复制模板包
- 站点配置冲突规则
- 接入检查清单
- 回滚与回放验收口径

本版不包含：
- 第二真实站点上线
- M10 Agent 价值验证

## 3. 正式接口

- `GET /api/v1/platform/station-governance/stations/:stationId/copy-package`
- `GET /api/v1/platform/station-governance/stations/:stationId/onboarding-playbook`

## 4. 固定前提

- 主样板站：`MME`
- 模板对照站：`RZE`
- `RZE` 只用于模板对比，不代表第二真实试运行站
- `warning` 允许带入接入完成态，但必须有人工作确认
- 回滚口径固定为：`模板 + 配置级回滚`

## 5. 接入步骤

1. 绑定站点复制模板包
2. 补齐最小接入单元
3. 执行接入检查清单
4. 按 `MME` 生产试运行 SOP 回放最小导入链
5. 生成正式接入验收记录

## 6. 冲突规则最小集

### 阻断型

1. 站点主标识冲突
2. 控制层级冲突
3. 关键审计开关冲突
4. 正式导入链字段映射冲突
5. `blocked` 质量门槛冲突

### Warning 型

1. 报表展示文案差异
2. 默认班组映射差异
3. 默认可见角色差异
4. 非关键提示级别差异
5. 资源重复绑定待清理

## 7. 接入检查清单最小集

1. 基础身份
   - 站点主记录已建立
   - 控制层级已定义
   - 站点模板已绑定
2. 运行对象
   - 班组已配置
   - 区域已配置
   - 设备最小集已配置
3. 数据链路
   - 正式导入链可用
   - 样板站对象链字段可映射
   - `qualityChecklist` 能返回 `clear / warning / blocked`
4. 报表与审计
   - 平台日报可出
   - 站点日报可出
   - 审计链可回放
5. 风险门槛
   - warning 项已列出并确认
   - blocked 项已清零

## 8. 回滚与回放验收口径

### 回滚对象

- 站点模板绑定
- 站点配置覆盖项
- 班组 / 区域 / 设备映射
- 报表展示与治理开关

### 不回滚对象

- 已落库的真实业务导入数据
- 审计事件
- 已形成的日报结果

### 回放验收

- 参考样板：`MME`
- 参考文档：[Sinoport_OS_MME_生产试运行SOP_v1.0.md](/Users/lijun/Downloads/Sinoport/docs/Sinoport_OS_MME_生产试运行SOP_v1.0.md)
- 必须验证：
  - 正式导入链回放成功
  - 对象链回读完整
  - 平台日报 / 站点日报 / 质量检查表回读正常
  - 审计链可追溯

## 9. 接入完成判定

以下条件同时成立才算接入完成：

1. 没有未解释的强制一致项冲突
2. 接入检查清单没有 `blocked`
3. warning 已在验收记录中完成确认
4. 回滚方案已准备
5. 导入回放可重复执行
6. 已形成正式接入验收记录
